##2. 새로 배운점
---
```
*MongoDB는 node.js와 같은 V8 엔진을 사용하기 때문에 
JavaScript 코드로 디비를 관리 할 수 ​​있다.

*테이블 형태로 저장하는 관계형 DB와 달리 NoSQL는 Document key : value 형태로 
저장. (BSON구조) 데이터를 일단 메모리에 저장했다가 순차적으로 주기억장치에 반영. 
같은 DB를 복제해 여러 군데에 두거나 하나의 데이터를 여러 곳에 분산해 저장하는 
분산 처리 방식을 사용해 안전성을 확보

*MongoDB는 창에서 다룰 수있는 나침반으로 큰 사이즈의 파일도 한꺼번에 
나타나는 수정, 삭제 등이 가능

```
