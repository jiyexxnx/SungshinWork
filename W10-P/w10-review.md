```
1. 웹 프로그래밍 언어에 SQL삽입
2. SQL 전용 언어 사용하기.
3. 일반 프로그래밍 언어에 SQL삽입
4. 데이터베이스 관리 기능과 비주얼 프로그래밍 기능을 갖춘 
    GUI 기반 SW개발도구 사용
```

